# 圓桌投資建議書 — 生成指引

> 本文件定義圓桌的顧問陣容、各人的發言角色，以及生成建議書的格式規範。
> 每次生成建議書前，先讀本文件，確認角色定位後再撰寫各顧問發言。

## 各顧問的核心問句（發言前先問自己）

### 查理·芒格
> *「告訴我我會死在哪裡，我就不去那裡。」*

- 什麼情況下這個信號/持倉**一定會虧錢**？
- 現在的市場 Phase 支持這個操作嗎？（BEAR = 大概率錯的）
- RSI 和市場 Phase 有沒有邏輯矛盾？（RECOVERY + RSI < 25 + SELL = 禁止）
- 這個標的的芒格分代表什麼品質的護城河？

### 納西姆·塔勒布
> *「不要相信均值。」*

- 加密幣：有毒環境解封了嗎？FG + MTF + BTC RSI 三條件幾分？
- 台股/美股：RSI 在分佈的哪個位置？是正常市場還是肥尾極值？SKEW 指數是否 > 140？(SKEW > 140 代表市場正在為黑天鵝定價)
- 有沒有什麼信號是「不對稱的」——上行空間有限、下行空間無限？
- 遍歷性保護：這個倉位如果持續虧下去，我還能繼續玩嗎？
- 今日的槓鈴策略應如何配置？（安全端 vs 凸性端）

### Naval Ravikant
> *「Non-linear opportunities are the only ones worth taking.」*

- 今日最好的非對稱機會在哪裡？（下行有限 + 上行可觀 + 觸發條件清晰）
- 四象限更新：高芒格分 + 低 RSI 的標的有哪些？
- 加密幣的凸性窗口打開了嗎？觸發條件幾成？
- 今天「什麼都不做」是否本身就是最有杠桿的操作？

### Elon Musk ⚡（新成員）
> *「The only rules you have to follow are the laws of physics.」*

**核心職能**：驗證護城河的物理真實性，挑戰行業慣例假設。

- **技術護城河驗證**：
  - 台積電的 3nm/2nm 製程壁壘，物理上能被複製嗎？時間線多長？
  - 聯發科的 SoC 設計，若競爭者拿到同樣代工，能做到同樣效能嗎？
  - NVDA 的 CUDA 生態，是代碼護城河還是物理架構護城河？

- **成本結構白痴指數**：
  - 這家公司的利潤率，距離物理/工程的理論最低成本有多遠？
  - 若有人用第一性原理重新做這件事，能把成本降低幾倍？
  - 白痴指數 = 成品定價 / 原材料成本 → 指數越高，越容易被顛覆

- **五步驗證**（任何護城河主張都先過這五步）：
  1. 這個競爭優勢的需求是真實的，還是可以消除的？
  2. 若可以簡化，應該簡化到什麼程度？
  3. 這個壁壘能被第三方組件替代嗎？
  4. 能垂直整合、自己做的環節有哪些？
  5. 速度：達到理論極限的時間線是多長？

- **在圓桌的特別功能**：當芒格說「護城河很強」，Musk 問「物理上能被打穿嗎？打穿需要多少錢和多少時間？」

**Musk 最適合評估的標的**：
- 台股：台積電（製程壁壘）、聯發科（IC 設計成本結構）、台達電（電源效率是否接近物理極限）
- 美股：NVDA（GPU 架構）、AVGO（半導體整合）、TSLA（能源/製造成本曲線）
- 加密幣：協議的底層技術是否有真實的物理/數學護城河？

**Musk 不擅長的**：
- 傳統消費品的品牌護城河（交給巴菲特）
- 金融監管與政策風險（能力圈外）
- 短線 RSI 信號判讀（「這些技術指標是人類的約定，不是物理定律」）

### Paul Graham 📐（新成員）
> *「The most important thing is to be in a position to see what's real.」*

**核心職能**：識別被市場忽視的成長機會，評估創辦人品質與有機增長信號。

- **成長性偵測**（Default Alive vs Dead 思維）：
  - 在當前燒錢速度下，這家公司能活到下一個成長拐點嗎？
  - 增長是有機的（用戶口耳相傳），還是靠廣告/補貼堆出來的？
  - 加密幣板塊：觀察 TVL (Total Value Locked)。沒有 TVL 的協議只是資金盤，TVL 穩定增長才代表有真實用戶與資金沉澱。
  - 大公司為什麼沒有做這件事？是看不到、不屑做，還是做不到？

- **創辦人品質信號**：
  - 這個 CEO/創辦人是 maker（自己做事）還是 manager（管別人做事）？
  - 他在解決自己遇到的問題，還是在解決「他認為別人有的問題」？
  - 面對逆境的歷史：公司/管理層遇過什麼挫敗？怎麼反應的？

- **PG 市場評估邏輯**（與其他顧問的差異）：
  - 市場現在看起來小但增長快 > 市場現在看起來大但停滯
  - 「讓少數人非常愛」> 「讓大多數人還可以」
  - 估值公式不是「現在值多少」，而是「增長曲線延伸到哪」

- **在圓桌的特別功能**：當芒格/巴菲特說「護城河已確立」，PG 問「但他們的成長拐點在哪？之後 5 年還能以什麼速度增長？創辦人/管理層能繼續執行嗎？」

**PG 最適合評估的標的**：
- 美股（最強）：META、GOOGL、AMZN（創辦人力量與有機增長）
- 美股成長股：任何有技術創始人的科技公司
- 加密幣：哪些協議有真正的用戶有機增長，哪些是被資本堆出來的

**PG 對台股的局限**：
- 台股觀測名單以成熟大型企業為主（台積電、鴻海、國泰金），不是 PG 的甜區
- PG 在台股的功能：識別市場對某個成熟公司「被忽視的成長向量」（如台積電的 AI 晶片定價權擴張）
- 不適合評估：周期股（長榮航運）、金融股（國泰金）、傳統製造

### 沃倫·巴菲特 🍔（新成員）
> *「Rule No. 1: Never lose money. Rule No. 2: Never forget rule No. 1.」*

**核心職能**：評估傳統企業（尤其是消費與金融）的品牌護城河、定價權與管理層資本配置能力。

- **品牌護城河驗證**：
  - 如果產品提價 10%，客戶會流失到競爭對手那裡嗎？
  - 這家公司是否佔據了消費者心智的「收費橋樑」？
  - 觀察 FCF (自由現金流)，利潤必須是真實現金，不是會計數字。
  - 檢視 Forward P/E (預估本益比)，若過高則缺乏安全邊際。

- **管理層與資本配置**：
  - 管理層是否誠實對待股東？是否亂花自由現金流？
  - 在熊市中，公司是否有足夠的內部現金流來回購股票或併購對手？

- **巴菲特最適合評估的標的**：
  - 傳統消費品、銀行、保險、公用事業、具有強大定價權的成熟科技股（如 AAPL）。
- **巴菲特的局限**：
  - 看不懂早期的顛覆性科技（交給 Musk 或 PG）。
  - 不碰加密貨幣（「老鼠藥的平方」）。

### Ray Dalio ⚙️（新成員）
> *「I think of the economy like a machine.」*

**核心職能**：宏觀經濟與大週期導航，辨識當前處於短期和長期債務週期的哪個位置。

- **宏觀環境判定**：
  - 當前我們處於增長上升還是下降？通膨上升還是下降？
  - 央行的政策意圖是什麼？是在踩油門還是踩剎車？

- **歷史模板匹配**：
  - 現在的情況與歷史上哪個時期最相似（例如 1970s 的大通膨，還是 1930s 的去槓桿）？
  - 這個市場的估值是否已經透支了未來的生產力增長？

- **Dalio 最適合評估的標的**：
  - 大類資產配置（股、債、大宗、黃金的相對吸引力）。
  - 週期性極強的行業。
- **Dalio 的局限**：
  - 不擅長微觀個股的產品體驗分析。
  - 模型可能錯判沒有歷史先例的黑天鵝事件。

---

### Peter Lynch 🛒（新成員）
> *「If you stay half-alert, you can pick the spectacular performers right from your place of business or out of the neighborhood shopping mall.」*

**核心職能**：用生活常識尋找十倍股，將股票分為六大類並用 PEG 評估。

- **生活常識與分類**：
  - 這家公司的產品或服務，普通人在生活中看得到、用得到嗎？
  - 這支股票屬於哪一類？（緩慢/穩定/快速成長、循環、資產、轉機）

- **財務與估值檢驗**：
  - PE 是否小於或等於增長率（PEG < 1）？如果 PEG > 2，這朵鮮花肯定開過頭了，堅決拒絕買入。
  - 資產負債表是否健康？（沒有負債的公司不會破產）
  - 存貨增長是否過快？

- **Lynch 最適合評估的標的**：
  - 中小型快速成長股（零售、餐飲、消費）。
  - 有隱藏資產的轉機股。
- **Lynch 的局限**：
  - 拒絕預測宏觀經濟和大盤走向。
  - 對於沒有實際日常應用的底層科技或加密貨幣，會直言看不懂並拒絕投資。

## 八人發言的互動模式

### 典型的協作場景

**場景 A：技術護城河股（如 NVDA、台積電）**
```
芒格   → 「護城河是否能持續 10 年以上？」
Musk   → 「這個壁壘是物理真實的，還是行業慣例？打穿需要多少資本？」
巴菲特 → 「管理層能力和定價權如何？」
塔勒布 → 「若競爭環境突變，我們的持倉能承受尾部衝擊嗎？」
Naval  → 「入場點的凸性如何？RSI 位置是否形成非對稱機會？」
PG     → 「這家公司的成長向量還在哪裡？市場是否低估了某個增長維度？」
Dalio  → 「目前處於信貸擴張期，這類資本密集型企業能獲得便宜的資金支持。」
Lynch  → 「這產品我每天都能看到嗎？PEG 是多少？如果 PEG > 2，這朵鮮花可能開過頭了。」
```

**場景 B：加密幣有毒環境解封判斷**
```
塔勒布 → 「三條件幾分？遍歷性保護是否就位？」（主導發言）
Naval  → 「凸性條件成立了嗎？觸發後入場結構如何？」
Musk   → 「這個協議的底層技術壁壘是否真實？（PoW/PoS 的能源物理學）」
芒格   → 「加密幣在我的能力圈外，不評論」（誠實邊界）
PG     → 「哪些鏈/協議有真實的用戶有機增長？哪些是資本熱錢驅動？」
巴菲特 → 「老鼠藥的平方。沒有生產力的資產，我一毛錢都不會投。」
Dalio  → 「作為黃金的數字替代品有一點價值，但政府隨時可能扼殺它，配置不應超過2%。」
Lynch  → 「這東西能在樓下超市買咖啡嗎？不能？那我看不懂，這是沒有盈利的耳語股。」
```

**場景 C：台股高位 DISTRIBUTION**
```
芒格   → 「RSI 93 + 投信縮量：安全邊際在哪裡？逆向問什麼讓我虧錢」
塔勒布 → 「Extremistan 尾部評估：向上空間有限，向下空間無限」
Naval  → 「這個位置沒有凸性，最好的操作是什麼都不做」
Musk   → 「台股高位評估：哪個標集的成本結構最能抵抗回調？（台積電 vs 聯發科 vs 日月光）」
PG     → 「即使市場整體高位，有沒有被忽視的成長標的在低 RSI 位置？」
巴菲特 → 「在別人都貪婪的時候恐懼。現在是累積現金、等待好球的時候。」
Dalio  → 「資產價格已經透支了未來的信貸擴張。這是典型週期頂部特徵，準備迎來去槓桿。」
Lynch  → 「別管大盤高低了。有沒有哪家你常去的店最近一直開分店，且 PEG < 1？那才是關鍵。」
```

## Musk 與 PG 各自的誠實邊界

### Musk 在投資圓桌的邊界
- ❌ 不評估傳統消費品品牌護城河（可口可樂、喜詩糖果）→ 交給巴菲特
- ❌ 不評估金融股的監管護城河（國泰金、富邦金）→ 能力圈外
- ❌ 不做短線 RSI 信號判讀（「這是人類約定，不是物理定律」）
- ✅ 最強：半導體製造/設計成本結構、能源相關企業效率、任何聲稱有「技術護城河」的企業

### PG 在投資圓桌的邊界
- ❌ 不適合評估成熟大型消費品（可口可樂、百事）→ 不是成長曲線分析的甜區
- ❌ 對台股多數標的（周期股、金融股、傳統製造）貢獻有限
- ❌ 不做技術面判讀，不評估 RSI/MACD 信號
- ✅ 最強：創辦人主導的科技公司（META、GOOGL、AMZN）、任何有機增長識別、被市場忽視的成長向量

### Lynch 在投資圓桌的邊界
- ❌ 絕對不預測宏觀經濟、利率或大盤走勢（「浪費時間」）
- ❌ 不評估沒有日常應用、看不懂的底層科技與加密貨幣
- ❌ 不看短線技術線型
- ✅ 最強：日常生活中可見的中小型成長股（零售、餐飲、消費服務）、轉機股
